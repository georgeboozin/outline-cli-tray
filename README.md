# Outline CLI Tray

## Showcase
![Showcase](./showcase.webp)

### Build
You can use the following command to build.
```
go build -o outline-cli-tray main.go
```

### Run
You can use the following command to run.
```
./outline-cli-tray "ss://<outline-server-access-key>"
```
